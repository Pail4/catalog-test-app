## Каталог онлайн-курсов

Тестовое приложение каталога онлайн-курса на чистом JS

Деплой [github pages](https://pail4.github.io/catalog-test-app/)
Деплой [vercel](https://catalog-test-app.vercel.app/)

### Задача
Сверстать и оживить небольшой блок точно по [макету](https://www.figma.com/design/iuXLXifXgv4ZCzad0KeYKr/Test-Task?node-id=0-1&p=f&t=KsNXMs8WQDm1CVOJ-0) в Figma.

Это типичный карточный каталог с поиском и фильтрами — то, с чем нужно работать каждый день и не только :)

Что нужно сделать:
- Pixel-perfect вёрстка по макету (1920 px → до мобильных 320 px)
- Адаптив «резиновый» (без жёстких брейкпоинтов, плавно масштабируется)
- Поиск по названию карточек (живой, без перезагрузки)
- Фильтрация по категориям (всё сгруппировано по категориям)
- Чистый и понятный код по БЭМ
- Стек: HTML + JavaScript (vanilla) + SCSS
- Никаких библиотек и фреймворков — только нативный JS
